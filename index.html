Welcome to the User Authentication System

This system allows users to securely sign up, log in, and access protected routes. Features include:

  • Secure user registration with hashed passwords
  • Secure login with session management
  • User role-based access control
  • Protected routes and functionalities
  • Profile management and password change functionality

Use the navigation bar above to sign up for a new account or log in if you already have an account.
